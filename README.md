Controls for the arcade machine:

move-> joystick

fire ball-> blue

jump-> green

pause menu-> red

change powers-> yellow

use item-> white


[If you want to play online :)](https://rosabm.github.io/GMD/) Play it in full screen!

Controls for a normal keyboard:

move-> arrow keys

fire ball-> z

jump-> x

pause menu-> v

change powers-> space bar

use item-> b
